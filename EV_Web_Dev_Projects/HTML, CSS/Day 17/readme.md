# Animations
## Transitions
- It starts only whne some event occurs like hover
- We cannot specify the intermediate state

Animation in CSS Keyframes

# Syntax
    @keyframes identifier {
        to{}
        from{}
    }

# Example 

@keyframes anim {
    0% {color:black}
    100% {color:white; background-color:black;}
}

# Animation Property
animation-name
animation-duration
